# SchrodingerSolver.jl

Pakiet w języku Julia do numerycznego rozwiązywania jednowymiarowego równania Schrödingera. 

## Grupa
* Julia Zalewska 
* Oliwia Marut 
* Natalia Chmiel 

## O projekcie
Pakiet implementuje i porównuje dwie metody numeryczne:
1. Metodę Różnic Skończonych (FDM)
2. Metodę Numerowa

Cała analiza matematyczna, założenia i złożoności algorytmów oraz wizualizacje wyników znajdują się w pliku `math.ipynb`.

## Uruchomienie
Projekt jest pakietem w języku Julia. Aby zapoznać się z wynikami naszej pracy:
1. Sklonuj repozytorium.
2. Otwórz i uruchom notatnik `math.ipynb` (wymagane pakiety to m.in. `LinearAlgebra` i `SparseArrays`).
